# Mini Calendar

A simple mini calendar built with HTML, CSS, and JavaScript that displays the current date, day, month, and year in a clean card-style layout.

## Live Demo
Check it live here: [Mini Calendar](https://asma-ehsan.github.io/Mini-Caldender/)

## Features
- Displays current date, day, month, and year.
- Automatically updates using JavaScript `Date` object.
- Clean, responsive, and beginner-friendly layout.

## Technologies Used
- HTML
- CSS
- JavaScript

## How It Works
- JavaScript fetches the current date using `new Date()`.
- Day and month names are mapped using arrays.
- Values are dynamically inserted into HTML elements using `innerHTML`.

## What I Learned
- Using the JavaScript `Date` object to get real-time data.
- Updating DOM elements dynamically.
- Styling layouts with CSS Flexbox.
- Importance of building small projects to strengthen basics.
